Searching and Registering Manually

This topic describes how to find and register with the network manually.

Procedure

  1. Choose Settings > Network.
  2. Select Manual search and register.
  3. Click Refresh.
  4. Select the network to be registered with from the Operator Name list.
  5. Click Register.
