Selecting the Network Type

Select the network type of the network service provider when searching for its network, so that you can find the network and log in to it rapidly.

Context

NOTE:

Contact your network service provider for the network type and frequency band.

Procedure

  1. Choose Settings > Network.
  2. Select the corresponding network type in the Network Type drop-down list box.
  3. Select the corresponding frequency band in the Band drop-down list box.
  4. Select the corresponding frequency LTE band in the LTE Band drop-down list box.
  5. Click OK to save and validate the settings.
