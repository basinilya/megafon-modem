Deleting a Dial-Up Profile

This topic describes how to delete a dial-up profile.

Procedure

  1. Choose Settings > Profile.
  2. Select a profile from the Profile name list.
  3. Click Delete to display the MegaFon dialog box.
  4. Click Yes to delete the dial-up profile.
