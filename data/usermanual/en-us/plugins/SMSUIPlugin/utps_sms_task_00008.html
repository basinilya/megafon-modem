Setting the Text Message Prompt

If the text message prompt is set, an alert tone is played or a visual prompt is displayed when there is an incoming text message.

Procedure

  1. Choose Settings > SMS-Client.
  2. Select the audio prompt or visual prompt.
    Option Description

    Visual prompt

    Select the Show a notification when a new text message arrives check box. When there is an incoming text message, a prompt dialog box is displayed. You have the following options:

    • Click View to view the text message.
    • Click Cancel to view the text message later.

    Audio prompt

    Select the Play a sound when a new message arrives check box and select an alert tone. When there is an incoming text message, the alert tone is played.

    1. Click Browse to select the alert tone file.
    2. Click Test to play the alert tone. Click Stop to stop playing the alert tone.
