Enabling and Disabling PIN Verification

When the PIN verification is enabled, next time you launch the terminal device, you need to enter the PIN. When the PIN verification is disabled, next time you launch the terminal device, you do not need to enter the PIN.

Context

  • Choose Settings > General, and then select Activate in the Check PIN code area.
  • Choose Settings > General, and then select Deactivate in the Check PIN code area.
